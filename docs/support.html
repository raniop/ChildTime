שאלות נפוצות

איך מחברים מכשיר של ילד?
במכשיר ההורה: כרטיס הילד → "חַבְּרוּ אֶת הַמַּכְשִׁיר" → מציג קוד QR. במכשיר הילד בוחרים "מכשיר של ילד" וסורקים את הקוד.

שכחתי את קוד ההורה.
הקוד נקבע בכניסה הראשונה להגדרות. אפשר לשנותו בהגדרות הורה ← קוד.

איך בוחרים אילו אפליקציות נחסמות?
במכשיר הילד, בהגדרות הורה ← "אפליקציות לחסום" (נדרשת הרשאת Screen Time).

איך מוחקים נתונים?
הגדרות הורה ← פרטיות ← מחיקת נתונים, או מחיקת ילד/ה מהכרטיס בלוח הבקרה.
