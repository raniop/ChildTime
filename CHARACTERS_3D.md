# מודלי דמויות תלת-ממד — מה לספק

כדי שהדמויות יהיו "אמיתיות" כמו רובלוקס/ברולסטארס, צריך קבצי מודל 3D אמיתיים.
ברגע שתפיל קבצים לפי המפרט הזה — אחבר אותם ל-`Character3DView` ואאמת בסימולטור.

## איפה לשים
שים את הקבצים בתיקייה:

```
ChildTime/Characters3D/
```

(הפרויקט משתמש ב-Xcode 16 synchronized groups — כל קובץ בתיקייה הזו ייכלל אוטומטית ב-bundle. לא צריך לגעת ב-Xcode.)

אחרי שתשים — **תגיד לי את שמות הקבצים** ואני מחבר אותם.

## פורמט
- **`.usdz`** (מועדף — מכיל טקסטורות מוטמעות) או `.scn`.
- מודל של **דמות אחת** לכל קובץ.

## דרישות
| | |
|---|---|
| **כיוון** | הדמות עומדת זקופה (ציר Y למעלה), פנים לכיוון הצופה (+Z). |
| **קנה מידה** | לא משנה — הקוד מנרמל אוטומטית לגובה אחיד וממרכז. |
| **טקסטורות** | מוטמעות בקובץ (usdz עושה זאת אוטומטית). |
| **פוליגונים** | סביר למובייל (עד ~50K) כדי לרוץ חלק על אייפד. |
| **Rigging** | לא חובה לשלב הנוכחי (תנוחה סטטית מספיקה). נדרש בהמשך לאנימציות. |

## מאיפה משיגים
- **Mixamo** (Adobe, חינם) — דמויות rigged, ייצוא ל-FBX→המרה ל-usdz (דרך Reality Converter של אפל, חינם).
- **Sketchfab / TurboSquid / Unity Asset Store** — חבילות דמויות (חינם ובתשלום), חפש "stylized character usdz".
- **Reality Converter** (אפל, חינם) — ממיר OBJ/FBX/glTF ל-usdz.

## דוגמה לשמות
```
ChildTime/Characters3D/
  ninja.usdz
  robot.usdz
  princess.usdz
```
ואז: `Character3DView(modelName: "ninja.usdz")`.
